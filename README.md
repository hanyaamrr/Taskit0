# Taskitoo - Task Manager 🚀

Taskitoo is a full-stack task management system built with Spring Boot. It includes secure authentication and allows users to manage their tasks safely.

---

## ✨ Features

- User authentication (JWT + Google OAuth2) 🔐
- Create, update, and delete tasks
- AES encryption for sensitive data
- Basic security protections (CORS, rate limiting)

---

## 🛠️ Tech Stack

- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- JWT
- OAuth2 (Google)
- MySQL / PostgreSQL

---

## ⚙️ Setup

1. Clone the repository
2. Configure `application.properties`
3. Add required security keys (JWT, AES)
4. Run the Spring Boot application
